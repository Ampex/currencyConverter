### How to use QUERY PARAMETERS

Before using any params, be sure to insert a question mark '?'
Each parameter separate with an ampersand '&'
Avaiable query params:

- value
- from
- to

#### Example below

```
http://bkasperski.pl/currs/?value=100&from=USD&to=EUR
```
